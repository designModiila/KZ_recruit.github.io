Tablet / Mobile

테스트 단말 (Tablet, mobile)

국내외 모바일 기기 점유율이 가장 높은 순위에 따라 스마트폰 4종, 태블릿 2종에 대하여 테스트 진행
단말 확정 후 업데이트 예정

DTD / language TYPE / Encoding 선언

			<!DOCTYPE html>
			<html lang="ko">
			<head>
			<meta charset="utf-8">

viewport 선언

			<meta name="viewport" content="width=device-width, initial-scale=1, user-scalable=yes">
속성 설명 값
width viewport의 가로크기를 지정 device-width
initial-scale 페이지가 최초로 로드될 때 사용할 기본 줌인/줌아웃 값, 1은 100%를 의미 1(기본값) ~ 10
user-scalable 사용자에게 줌인/줌아웃을 허용할지 여부를 설정 yes(기본값) / no
